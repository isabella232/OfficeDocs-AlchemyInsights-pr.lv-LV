# <a name="sync-your-browsing-data-across-your-devices"></a>Pārlūkošanas datu sinhronizēšana visās ierīcēs

Jebkurā ierīcē izmantojiet Microsoft Edge sinhronizācijas funkcionalitāti, lai sakārtotu savu izlasi un turpinātu pārlūkošanu vietā, kur to neņēmāt.

## <a name="sync-your-browsing-data-on-a-desktop-computer"></a>Pārlūkošanas datu sinhronizēšana galddatorā

1. Microsoft Edge uzdevumjoslā atlasiet profila ikonu.
    
    - Ja esat jau pierakstījies, atlasiet Pārvaldīt **profila iestatījumus**.
    - Ja neesat pierakstījies, atlasiet **pogu Pierakstīties** un pierakstieties. Uznirstošajā **logā Profila** sinhronizēšana atlasiet **Pielāgot**.

1. Lapā **Jūsu profils** atlasiet **Sinhronizēt**.

1. Ieslēdziet vai izslēdziet sinhronizāciju atsevišķiem pārlūkošanas vienumiem. Ieslēdzot sinhronizēšanu **izlasei,** tiek sinhronizētas arī visas cilnes, ko iepriekš atlikāt malā.

Lai sinhronizētu cita konta datus, Microsoft Edge uzdevumjoslā atlasiet profila attēlu, pievienojiet vai atlasiet citu profilu un pēc tam atkārtojiet iepriekš norādītās darbības.

## <a name="sync-your-browsing-data-on-your-ios-or-android-device"></a>Pārlūkošanas datu sinhronizēšana iOS vai Android ierīcē

1. Ja mobilajā ierīcē neesat instalējis Microsoft Edge, dodieties uz lietojumprogrammas lejupielādes lapu un izpildiet norādījumus, lai instalētu programmu, atlasiet kontu un pēc tam pierakstieties.

1. Programmā Microsoft Edge atlasiet savu profila attēlu un pēc tam atlasiet **Sinhronizēt**.

1. Ieslēdziet sinhronizāciju un pēc tam atzīmējiet vai notīriet izvēles rūtiņas noteiktiem datu vienumiem, kurus vēlaties sinhronizēt.

Papildinformāciju skatiet rakstā [Pierakstīšanās, lai sinhronizētu Microsoft Edge dažādās ierīcēs](https://go.microsoft.com/fwlink/?linkid=2145501).
